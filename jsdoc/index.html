The IJavascript kernel is implemented by class Kernel in kernel.js and the message handlers in handlers_v4.js and handlers_v5.js.

The kernel makes use of package jmp to parse messages in the IPython/Jupyter messaging protocol, and package nel to provide Javascript REPL sessions. This package provides Javascript sessions by means of a Node.js server implemented in nel_server.js.
